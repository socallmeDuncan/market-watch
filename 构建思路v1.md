# A股盘中行情监控脚本 Spec

## 1. 项目名称

`market-watch`

## 2. 项目目标

为盘中交易决策提供一个轻量级行情快照工具，用于快速抓取协创数据、香农芯创及相关指数/板块的实时行情，并输出为 Markdown 或 JSON，方便直接复制给 ChatGPT 进行买卖判断。

本工具**不做自动交易、不自动下单、不替代人工决策**，只用于盘中观察和决策辅助。

---

## 3. 当前使用场景

用户当前主要关注：

1. 协创数据 `300857`
2. 香农芯创 `300475`
3. AI 硬件 / 算力 / 存储板块整体情绪
4. 创业板 / 半导体 / 科技成长风格是否同步修复或继续走弱
5. 协创是否出现：

   * 站回 300
   * 站上 305
   * 跌破 290 后拉回 292—295
   * 跌破 285 后是否继续弱
   * 270—280 是否出现承接
   * 240—260 主力成本区是否出现战略买点

## 4. 数据源选择

### 4.1 MVP 数据源：AKShare

使用 `AKShare` 作为第一版数据源。

AKShare 是基于 Python 的财经数据接口库，可以获取股票、指数、基金、期货、外汇、债券等数据。官方说明其目标是获取相对权威财经网站公布的原始数据，并进行数据采集、清洗和落地。AKShare 当前要求 Python 64 位 3.9 或以上版本，通用安装命令为：

```bash
pip install akshare --upgrade
```

### 4.2 主要接口

#### A股实时行情

优先使用：

```python
ak.stock_zh_a_spot_em()
```

用途：

* 获取沪深京 A 股实时行情
* 一次返回全部 A 股
* 再按股票代码筛选目标股票

#### 指数实时行情

优先使用：


## 10. 交易规则内置说明

脚本不直接给出“买入/卖出”的最终建议，但可以给出状态标签，方便用户发给 ChatGPT 判断。

### 10.1 协创数据关键价位

|      价位 | 含义           |
| ------: | ------------ |
|  305 以上 | 强修复确认区       |
|  300 以上 | 初步右侧确认区      |
| 292—300 | 震荡观察区        |
|  290 以下 | 短线弱势观察区      |
|  285 以下 | 破位风险区        |
| 270—280 | 深回调买回观察区     |
| 240—260 | 主力成本附近，战略观察区 |

### 10.2 状态判断

#### 状态 1：右侧确认

条件：

```text
协创最新价 >= 300
且 5分钟涨跌为正
且 未明显冲高回落
```

输出：

```text
协创初步站回 300，进入右侧确认观察，可考虑小仓位加仓。
```

#### 状态 2：强修复

条件：

```text
协创最新价 >= 305
且 香农芯创同步上涨
且 创业板/半导体方向不弱
```

输出：

```text
协创站上 305 且板块同步修复，短线承接较强。
```

#### 状态 3：震荡观察

条件：

```text
协创在 292—300 区间
且 未有效站上 300
且 未跌破 290
```

输出：

```text
协创处于震荡观察区，不宜急于加仓。
```

#### 状态 4：假跌破观察

条件：

```text
盘中低点 < 290
但 当前价 >= 292
```

输出：

```text
协创出现跌破 290 后拉回，可能是假跌破，需要观察是否能稳定在 292—295 上方。
```

#### 状态 5：短线弱势

条件：

```text
当前价 < 290
且 5分钟涨跌仍为负
```

输出：

```text
协创跌破 290 且暂未修复，短线偏弱，不宜左侧加仓。
```

#### 状态 6：破位风险

条件：

```text
当前价 < 285
```

输出：

```text
协创跌破 285，进入破位风险区，应等待更明确止跌信号。
```

## 11. 初版 Python 脚本草案

文件名：`watch.py`

```python
import json
from datetime import datetime
from pathlib import Path

import akshare as ak
import pandas as pd
import yaml


def load_config(path: str = "config.yaml") -> dict:
    with open(path, "r", encoding="utf-8") as f:
        return yaml.safe_load(f)


def ensure_dirs(config: dict) -> None:
    for key in ["markdown_path", "json_path", "log_path"]:
        output_path = config["output"].get(key)
        if output_path:
            Path(output_path).parent.mkdir(parents=True, exist_ok=True)


def fetch_a_stocks(stock_codes: list[str]) -> pd.DataFrame:
    df = ak.stock_zh_a_spot_em()
    df["代码"] = df["代码"].astype(str)
    return df[df["代码"].isin(stock_codes)].copy()


def fetch_indices(index_names: list[str]) -> pd.DataFrame:
    # 第一版只抓沪深重要指数
    df = ak.stock_zh_index_spot_em(symbol="沪深重要指数")
    if "名称" in df.columns:
        return df[df["名称"].isin(index_names)].copy()
    return df


def get_xiechuang_status(row: dict) -> dict:
    price = float(row.get("最新价", 0))
    low = float(row.get("最低", 0))
    five_min = row.get("5分钟涨跌", 0)

    try:
        five_min = float(five_min)
    except Exception:
        five_min = 0

    if price >= 305:
        status = "强修复"
        note = "站上 305，若板块同步修复，可视为强确认。"
    elif price >= 300:
        status = "右侧确认"
        note = "站回 300，进入初步右侧确认区。"
    elif low < 290 and price >= 292:
        status = "假跌破观察"
        note = "盘中跌破 290 后拉回，需要观察能否稳定在 292—295 上方。"
    elif price < 285:
        status = "破位风险"
        note = "跌破 285，进入破位风险区，不宜左侧加仓。"
    elif price < 290:
        status = "短线弱势"
        note = "跌破 290 且暂未修复，短线承接不足。"
    else:
        status = "震荡观察"
        note = "处于 292—300 区间震荡，暂不构成明确加仓点。"

    return {
        "price": price,
        "low": low,
        "five_min_change": five_min,
        "status": status,
        "note": note,
        "above_300": price >= 300,
        "above_305": price >= 305,
        "below_290": price < 290,
        "below_285": price < 285,
        "intraday_broke_290_and_recovered": low < 290 and price >= 292,
    }


def make_markdown(timestamp: str, stock_df: pd.DataFrame, index_df: pd.DataFrame, signals: dict) -> str:
    stock_cols = [
        "代码", "名称", "最新价", "涨跌幅", "今开", "最高", "最低",
        "昨收", "成交额", "换手率", "涨速", "5分钟涨跌"
    ]
    stock_cols = [c for c in stock_cols if c in stock_df.columns]

    index_cols = [
        "代码", "名称", "最新价", "涨跌幅", "今开", "最高", "最低", "成交额"
    ]
    index_cols = [c for c in index_cols if c in index_df.columns]

    md = []
    md.append("# 盘中行情快照")
    md.append("")
    md.append(f"时间：{timestamp}")
    md.append("")
    md.append("## 个股行情")
    md.append("")
    md.append(stock_df[stock_cols].to_markdown(index=False))
    md.append("")
    md.append("## 指数行情")
    md.append("")
    if len(index_df) > 0 and index_cols:
        md.append(index_df[index_cols].to_markdown(index=False))
    else:
        md.append("未获取到指数数据。")
    md.append("")
    md.append("## 自动摘要")
    md.append("")

    x = signals.get("xiechuang")
    if x:
        md.append(f"- 协创数据状态：**{x['status']}**。")
        md.append(f"- 当前价：{x['price']}，盘中低点：{x['low']}。")
        md.append(f"- 判断：{x['note']}")

    md.append("")
    md.append("## 给 ChatGPT 的问题模板")
    md.append("")
    md.append("> 根据以上行情，协创现在是否买入 100 股？是否继续等待站稳 300？是否需要卖出机动仓？")

    return "\n".join(md)


def main():
    config = load_config()
    ensure_dirs(config)

    timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")

    stock_codes = [item["code"] for item in config["stocks"]]
    stock_df = fetch_a_stocks(stock_codes)

    index_names = [item["name"] for item in config["indices"]]
    index_df = fetch_indices(index_names)

    signals = {}

    xiechuang = stock_df[stock_df["代码"] == "300857"]
    if not xiechuang.empty:
        row = xiechuang.iloc[0].to_dict()
        signals["xiechuang"] = get_xiechuang_status(row)

    markdown = make_markdown(timestamp, stock_df, index_df, signals)

    output = {
        "timestamp": timestamp,
        "stocks": stock_df.to_dict(orient="records"),
        "indices": index_df.to_dict(orient="records"),
        "signals": signals,
    }

    md_path = Path(config["output"]["markdown_path"])
    json_path = Path(config["output"]["json_path"])

    md_path.write_text(markdown, encoding="utf-8")
    json_path.write_text(json.dumps(output, ensure_ascii=False, indent=2), encoding="utf-8")

    print(markdown)


if __name__ == "__main__":
    main()
```

## 15. 风险与限制

1. AKShare 不是交易所官方实时行情源。
2. 数据可能有延迟、字段变化、接口不稳定。
3. 本脚本适合辅助人工判断，不适合高频交易。
4. 不要用脚本结果自动下单。
5. 盘中交易还需要结合：

   * 外盘 SOXX / SMH / MU / NVDA
   * 韩国 KOSPI / SK 海力士 / 三星
   * A股半导体、存储、算力板块整体走势
   * 协创自己的成交量和盘口承接
6. 如果行情源异常，应以券商软件或官方行情终端为准。


## 18. 最小可用结论

第一版不需要做复杂系统。

最重要的是：

```text
一条命令运行；
输出一段 Markdown；
复制给 ChatGPT；
根据 290 / 295 / 300 / 305 / 280 / 240 这些关键价位判断下一步。
```

MVP 的目标不是自动化交易，而是让盘中决策从“凭感觉”变成“看数据 + 看规则 + 再判断”。
